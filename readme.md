# React Based Password Generator

The task set was :
Given any L, N, X, C return an L long string containing N numeric characters, X non alphabetic chars and C capital letters.

This has been implemented as 2 functions using a React Frontend to control the values of L, N, X, and C

The page can be reached at: [https://Cleggatron.github.io/ReactPWGenerator](https://Cleggatron.github.io/ReactPWGenerator)
